# POiO - Restaurant Management System

## Overview

POiO is a restaurant management system designed to support daily business operations, including order management, inventory control, customer service workflows, and administrative processes.

The project was developed as a software engineering solution focused on improving operational efficiency and centralizing business information within a single platform.

## Key Features

* Order management
* Inventory tracking
* Product administration
* Operational reporting
* Administrative dashboard
